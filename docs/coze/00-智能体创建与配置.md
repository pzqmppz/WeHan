# 扣子智能体创建与配置

## 概述

智能体（Agent）是扣子平台的核心概念，是大语言模型 + 工具 + 业务流程的组合。

## 创建流程

### 1. 注册登录

1. 访问扣子官网 https://www.coze.cn
2. 点击右上角「登录扣子」
3. 通过手机号完成注册和实名认证

### 2. 创建智能体

**方式一：手动创建**
1. 点击左上角「⊕」
2. 选择「创建智能体」
3. 填写名称和功能介绍
4. 生成或上传图标
5. 点击「确认」

**方式二：AI 创建**
1. 点击左上角「⊕」
2. 选择「AI 创建智能体」
3. 描述你的需求
4. AI 自动生成智能体配置

## 运行模式

| 模式 | 说明 | 适用场景 |
|-----|------|---------|
| 单 Agent（LLM 模式） | 直接对话 | 简单问答 |
| 单 Agent（工作流模式） | 多步骤处理 | 复杂流程 |
| 多 Agents | 多角色协作 | 复杂场景 |

## 模型配置

### 支持的模型

- 豆包（默认推荐）
- 通义千问
- 文心一言
- DeepSeek
- ChatGPT
- 等

### 生成参数

| 参数 | 说明 | 推荐值 |
|-----|------|-------|
| 温度 (Temperature) | 随机性 | 0.3-0.6 |
| Top P | 多样性 | 0.8-0.95 |
| 上下文轮数 | 记忆长度 | 10-20 轮 |
| 最大回复长度 | 输出限制 | 根据需求 |

**温度建议：**
- 0.1-0.3：技术文档、法律文书
- 0.4-0.7：故事、摘要、对话
- 0.8-1.0：科幻、搞笑、广告

## Prompt 编写

### 结构化 Prompt

```markdown
# 角色
你是一个[角色描述]，专门[功能描述]。

## 技能
### 技能1：[技能名称]
1. [具体描述]
2. [具体描述]

### 技能2：[技能名称]
1. [具体描述]

## 限制
- [限制条件1]
- [限制条件2]
- [输出格式要求]
```

### 优化 Prompt

1. 在「人设与回复逻辑」中输入初版 Prompt
2. 点击「优化」按钮
3. AI 自动重构为结构化格式
4. 手动补充缺失约束

## 开场白设置

支持 Markdown 格式，可插入：
- 用户昵称
- 图片
- 链接
- 预置问题

```markdown
你好，{{user_name}}！👋

我是你的 AI 助手，可以帮你：
- 📝 生成内容
- 🔍 搜索信息
- 💡 解答问题

有什么可以帮你的吗？
```

## 调试与发布

### 调试
1. 在右侧预览面板测试
2. 检查回复是否符合预期
3. 调整 Prompt 或技能
4. 重复测试

### 发布
1. 点击「发布」按钮
2. 选择发布平台
3. 填写更新日志
4. 确认发布

## 长期记忆

开启「长期记忆」功能后：
- 自动总结对话记录
- 存储关键信息
- 后续对话可回忆历史

## 最佳实践

1. **Prompt 清晰**：明确角色、技能、限制
2. **技能匹配**：根据需求选择合适的技能
3. **充分测试**：发布前多轮测试
4. **迭代优化**：根据用户反馈持续改进
