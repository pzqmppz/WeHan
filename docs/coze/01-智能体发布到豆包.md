# 扣子智能体发布到豆包

## 概述

扣子（Coze）创建的智能体可以直接发布到豆包 App，让用户在豆包中使用你的 AI 助手。

## 发布条件

> **重要**: 如果智能体及工作流只使用了豆包模型，则智能体在豆包中是**公开**的发布状态，任何用户都可以检索到智能体并与其对话。

如果使用了豆包以外的其他大模型，可能无法分享到豆包。建议修改模型后再次发布。

## 发布流程

### 1. 创建智能体

1. 访问扣子官网 https://www.coze.cn
2. 点击左上角「登录扣子」，通过手机号注册登录
3. 点击页面左上角「⊕」，选择「创建智能体」
4. 填写智能体「名称」和「功能介绍」
5. 图标可选择「AI生成」或「本地上传」

### 2. 配置智能体

**人设与回复逻辑 (Prompt)**
```
# 角色
你是一个[角色描述]，专门[功能描述]。

## 技能
### 技能1：[技能名称]
1. [具体描述]
2. [具体描述]

### 技能2：[技能名称]
...

## 限制
- [限制条件1]
- [限制条件2]
```

**模型选择**
- 默认使用「豆包」模型
- 可切换为通义千问、文心一言等

**添加技能**
- 插件：调用外部 API
- 知识库：上传文档实现 RAG
- 工作流：多步骤流程编排
- 触发器：定时任务

### 3. 预览与调试

在右侧「预览与调试」面板中：
1. 输入测试问题
2. 观察响应是否符合预期
3. 返回调整 Prompt 或工作流

### 4. 发布到豆包

1. 点击右上角「发布」按钮
2. 勾选发布平台「豆包」
3. 填写发布记录（更新说明）
4. 点击「发布」完成上线

## 在豆包中分享智能体

1. 打开豆包应用
2. 找到你的智能体
3. 点击对话页面的配置按钮
4. 点击分享图标复制分享链接
5. 将链接发送给好友即可使用

## 常见问题

**Q: 为什么无法点击分享按钮？**

A: 智能体中使用了豆包以外的其他大模型。建议修改为豆包模型后重新发布。

**Q: 审核不通过怎么办？**

A: 检查内容是否含敏感词、头像是否侵权。微信渠道需确保服务号已完成认证。
