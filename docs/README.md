B 端采用 Next.js 16 + TypeScript + Ant Design 5 技术栈，详见 `/web` 目录。

## C 端核心功能技术映射

### 求职 Agent

| 功能 | 技术方案 | 参考文档 |
|-----|---------|---------|
| AI 面试模拟 | 工作流 + 实时语音 SDK | 02-工作流开发, 06-WebSocket实时语音SDK |
| 简历解析 | 文件上传 + 大模型 | 05-开放API |
| 岗位匹配 | 知识库 RAG | 03-知识库使用 |
| 评估报告 | 工作流 + 结构化输出 | 02-工作流开发 |
| 分享链接 | Link Reader 插件 | 04-插件开发 |

### 心理健康 Agent

| 功能 | 技术方案 | 参考文档 |
|-----|---------|---------|
| 情绪咨询 | Prompt + 对话 | 00-智能体创建与配置 |
| 情绪记录 | 变量 + 数据库 | 00-智能体创建与配置 |
| 实时倾诉 | 实时语音 SDK | 06-WebSocket实时语音SDK |

### B 端数据互通

| 功能 | 技术方案 | 参考文档 |
|-----|---------|---------|
| 获取 B 端岗位 | HTTP 请求 / API | 04-插件开发, 05-开放API |
