## 十、快速启动命令

```bash
# 启动开发服务器
cd web && npm run dev

# 重置数据库
cd web && npx prisma db push --force-reset && npx prisma db seed

# 构建生产版本
cd web && npm run build

# 启动生产服务器
cd web && npm start
```
