# 武汉市高校毕业生就业创业政策汇编

> **数据来源：** 武汉市人民政府、武汉市人力资源和社会保障局
> **当前时间：** 2026年2月28日
> **数据状态：** 已筛选，仅保留当前有效政策

---

## 一、落户政策（长期有效）

### 1.1 高校毕业生落户

**政策内容：**
高等学校专科及以上学历（经教育部认证）毕业生、年龄不满45周岁（博士研究生、硕士研究生不受年龄限制），可凭毕业证申请登记为武汉市常住户口，其配偶及未成年子女均可随迁落户。

| 项目 | 内容 |
|------|------|
| **学历要求** | 大专及以上学历（经教育部认证） |
| **年龄要求** | 不满45周岁（硕博不限年龄） |
| **随迁人员** | 配偶及未成年子女可随迁 |
| **有效期** | 长期有效 |

**办理渠道：**
- 湖北公安政务服务平台（线上24小时申请）
- 各区政务服务中心公安户籍窗口

---

### 1.2 积分落户

**有效期：** 长期有效

**基本条件：**
1. 拥有合法稳定住所（含租赁）
2. 持有有效期内的武汉市居住证
3. 参加武汉市社会保险（城镇职工基本养老保险）
4. 年龄未达法定退休标准（男60周岁/女55周岁以下）
5. 累计积分达到75分

**积分规则：**
| 加分项 | 分值 |
|--------|------|
| 本人/配偶产权住房 | 60分 |
| 父母/子女名下房产 | 40分 |
| 租赁住房（每年） | 6分，最高60分 |
| 社保缴纳（每年） | 6分，最高60分 |
| 18-30周岁 | 30分 |
| 31-40周岁 | 20分 |
| 41-45周岁 | 10分 |

## 二、住房补贴政策

### 2.1 人才租赁房租金减免

**政策内容：**
取得普通高校全日制大专以上学历、毕业6年以内、2022年6月7日起在汉就业创业并正常缴纳社会保险、家庭在汉无自有住房的高校毕业生，可申请人才租赁房租金减免。

| 学历 | 优惠政策 | 最高免租金额/月 | 有效期 |
|------|----------|-----------------|--------|
| 全日制博士研究生 | 免缴2年租金 | 2000元 | 当前有效 |
| 全日制硕士研究生 | 免缴1年租金 | 1500元 | 当前有效 |
| 其他符合条件毕业生 | 市场租金70% | - | 当前有效 |

**申请条件：**
1. 在汉就业创业并于2022年6月7日之后首次缴纳社会保险
2. 毕业6年以内
3. 家庭在汉无自有住房

**减免期限：** 累计不超过3年

---

### 2.2 青年人才之家

**政策内容：** 为来汉创业就业青年提供最长15天免费住宿

**申请条件：** 非武汉户籍的全日制本科及以上学历毕业生

**有效期：** 当前有效

---

## 三、就业补贴政策

### 3.1 一次性吸纳就业补贴（高校毕业生）

**政策内容：**
对毕业年度或离校两年内未就业高校毕业生到中小微企业就业的，按规定给予高校毕业生个人一次性就业补贴。

| 项目 | 内容 |
|------|------|
| **补贴标准** | 1000元/人 |
| **申请条件** | 毕业年度或离校2年内未就业高校毕业生；到中小微企业就业；签订1年以上劳动合同；缴纳1个月以上社会保险费 |
| **有效期** | 当前有效 |

---

### 3.2 高校毕业生社会保险补贴（灵活就业）

**政策内容：**
对毕业年度和离校2年内未就业高校毕业生灵活就业后缴纳的城镇职工基本养老保险费、基本医疗保险费按规定给予社会保险补贴。

| 项目 | 内容 |
|------|------|
| **补贴标准** | 两项险种现行补贴标准合计为673.19元/月/人 |
| **补贴期限** | 最长累计不超过2年 |
| **有效期** | 当前有效 |

---

## 四、创业扶持政策

### 4.1 高校毕业生一次性创业补贴

**政策内容：**
对初次办理注册登记，经营6个月及以上且带动就业2人及以上（含创业者本人）的高校毕业生（毕业5年内），给予一次性创业补贴。

| 项目 | 内容 |
|------|------|
| **补贴标准** | 5000元/人 |
| **申请条件** | 毕业5年内的高校毕业生；初次办理注册登记；经营6个月及以上；带动就业2人及以上 |
| **申请时限** | 在初创组织法人毕业5年内提出申请 |
| **有效期** | 当前有效 |

**咨询电话：** 蔡甸区人社局 027-84942392

---

### 4.2 初创组织吸纳就业补贴

**政策内容：**
毕业5年以内的高校毕业生，在市内初次创办小微企业、个体工商户等，在登记注册1年内招用人员且签订1年以上劳动合同并缴纳社会保险费的，给予补贴。

| 项目 | 内容 |
|------|------|
| **补贴标准** | 1000元/人，补贴总额不超过2万元 |
| **申请时限** | 与被吸纳人员签订劳动合同后6个月内提出申请 |
| **有效期** | 当前有效 |

**注意事项：** 与一次性吸纳就业补贴不重复享受

---

### 4.3 初创企业创业场地租金补贴

**政策内容：**
在校大学生及毕业5年以内的高校毕业生创办初创企业并在各类创业孵化基地外租用场地用于生产经营的，给予租金补贴。

| 项目 | 内容 |
|------|------|
| **补贴标准** | 500元/月，最高每年6000元 |
| **补贴期限** | 累计不超过3年 |
| **有效期** | 当前有效 |

---

## 五、2026年新政策

### 5.1 武汉市支持人工智能OPC创新发展若干措施

**发布日期：** 2026年2月14日

**施行日期：** 2026年3月18日起

**有效期：** **至2028年12月31日**

**核心内容：**

| 政策项 | 补贴/支持 |
|--------|----------|
| 算力费用补助 | 50%补助，最高20万元/年，期限3年 |
| 免费算力支持 | 每年不少于2000卡时 |
| 初创OPC创业资助 | 最高100万元 |
| 人才租赁房优惠 | 博士标准2年免租（每月不超过2000元） |
| 购房补贴 | 每平方米1000元，最高10万元 |
| 创业担保贷款 | 最高500万元，最长2年 |
| 典型应用场景 | 示范项目最高100万元资金支持 |

**适用对象：** 人工智能领域"一人公司"（OPC）- 由具备AI应用开发与整合能力的人才或微型团队创办的新型创业实体

---

### 5.2 武汉市多子女家庭购房补贴

**政策内容：**
符合国家生育政策的二孩、三孩家庭在全市范围内购买新建商品住房的，发放购房补贴。

| 家庭类型 | 补贴标准 | 有效期 |
|----------|----------|--------|
| 二孩家庭 | 6万元 | **至2026年12月31日** |
| 三孩家庭 | 12万元 | **至2026年12月31日** |

**申请方式：** 通过"安居链"APP申请

**申请条件：**
- 二孩家庭：夫妻2016年1月1日（含）以后生育二孩
- 三孩家庭：夫妻2021年5月31日（含）以后生育三孩

## 九、注意事项

1. **政策叠加：** 同一人员只可享受一次生活补贴，享受本补贴的后续不再享受购房补贴
2. **离职处理：** 如在补贴发放期内离职并离开新区的，终止后续补贴发放
3. **诚信申报：** 弄虚作假者将停止发放补贴、追回已发补贴，永久取消享受政策资格
4. **求职陷阱防范：** 注意防范虚假招聘、黑职介、乱收费、培训贷、扣证件等求职陷阱
