| 功能 | 状态 | 说明 |
|-----|------|------|
| **AI 模拟面试** | ✅ 已实现 | 选择岗位 → 回答题目 → 生成评估报告 |


1. 前端开发工程师
2. 后端开发工程师
3. AI 算法工程师

### 开放 API

| 端点 | 方法 | 说明 |
|-----|------|------|
| `/api/open/jobs` | GET | 获取岗位列表 |
| `/api/open/policies` | GET | 获取政策列表 |
| `/api/open/interviews` | GET/POST | 查询/创建面试 |
| `/api/open/interviews/:id` | PATCH | 更新面试进度 |

## 技术架构

### C 端
- 扣子智能体 + 3 个工作流
- Chat SDK 网页嵌入
- HTTP 插件调用 B 端 API

### B 端
- Next.js 16 + TypeScript
- Ant Design 5 + Tailwind CSS
- PostgreSQL + Prisma
- NextAuth.js 认证
