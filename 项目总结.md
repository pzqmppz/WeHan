# WeHan 求职助手 - 项目总结

## 一、项目概述

**项目名称**：WeHan 求职助手 / 才聚江城

**定位**：为武汉高校毕业生提供一站式求职服务的智能平台

**架构**：双端产品（C端 + B端）

---

## 二、C 端 - WeHan 求职助手

**目标用户**：求职者（武汉高校毕业生）

**访问方式**：网页嵌入 AI 智能体（Chat SDK）

**核心功能**：

| 功能 | 说明 |
|-----|------|
| **AI 模拟面试** | 选择岗位，进行面试练习，获取评估报告 |
| **岗位查询** | 搜索武汉地区热门岗位信息 |
| **政策查询** | 了解武汉人才政策、就业补贴、落户优惠 |
| **心理疏导** | 缓解求职焦虑，提供情绪支持 |

**技术实现**：
- 扣子智能体 + 工作流
- Chat SDK 嵌入网页（支持工具、插件、HTTP 调用）

---

## 三、B 端 - 才聚江城管理平台

**目标用户**：企业、学校、政府、管理员

**访问方式**：Web 管理后台

**核心功能**：

| 模块 | 功能 |
|-----|------|
| **企业端** | 岗位管理、人才库、投递管理、面试安排 |
| **政府端** | 留汉指数、政策管理、数据统计 |
| **学校端** | 就业看板、学生管理、岗位推送 |
| **管理员** | 用户管理、企业管理、系统设置 |

**技术实现**：
- Next.js 16 + TypeScript
- Ant Design 5 + Tailwind CSS
- PostgreSQL + Prisma
- NextAuth.js（4角色 RBAC）
